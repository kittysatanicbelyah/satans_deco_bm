Ever wanted cooler stained glass like those in churches? You're on the right address, soul!

![like... actual stained glass](https://raw.githubusercontent.com/kittysatanicbelyah/random_bu-shi_go/refs/heads/main/deco/vpack1/bm_vpack1banner2.png)

This is an addon for Bars Mayhem, containing Vitrage blocks.

Vitrage also known as artistic stained glass. It was inspired mainly by churches stained glass.

- 30 variations of vitrages;
- Gold, Emerald, Copper (including oxidation process and waxed) variations;
- Variations for made up materials: Cloud and Dark Metal;
- Vitrage panels reminiscent of panels from Bars Mayhem. 3 for each material!

What can you do?

- Build something cool;
- Put dye in it by right click;
- Displace dye by right click with empty hand or with redstone signal.

You need Bars Mayhem to run it:
Check out releases on [github](https://github.com/kittysatanicbelyah/satans_deco_bm/releases) or download on [curseforge](https://www.curseforge.com/minecraft/mc-mods/satans-deco-bm).

Vitrage Pack 1.0.0 is supported by Bars Mayhem 1.0.0+
Vitrage Pack 0.1.0 is supported by Bars Mayhem 0.3.1

# About Satan's deco
