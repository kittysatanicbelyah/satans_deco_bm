![more bars, MORE bars, EVEN MORE bars](https://raw.githubusercontent.com/kittysatanicbelyah/random_bu-shi_go/refs/heads/main/deco/bm/bmbanner2.png)

This mod adds a lot of new variations of Iron Bars for 1.20.1. 
- 30 new variations;
- Gold, Emerald, Copper (including oxidation process and waxed) variations of barses;
- Barses variations for made up materials: Cloud and Dark Metal;
- Panel blocks, which you can rotate with your hand or redstone. 3 for each material.

# About Satan's deco

This is a multi-mod project to create a lot of somewhat unique and interesting decorations.

For simplicity all mods of this project are called in same fashion:
```java
"Satan's deco : *enter_name*"
```
