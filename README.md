# Satan's deco: Bars Mayhem - Vitrage Pack
## About Satan's deco

This is a multi-mod project to create a lot of somewhat unique and interesting decorations.

For simplicity all mods of this project are called in same fashion: 
```java
"Satan's deco : *enter_name*"
```

## About Bars Mayhem

This part of Satan's deco adds a lot of new variations of Iron Bars. 
- 30 new variations;
- Gold, Emerald, Copper (including oxidation process and waxed) variations of barses;
- Barses variations for made up materials: Cloud and Dark Metal.

## About Vitrage Pack

This is an addon for Bars Mayhem, containing Vitrage blocks. 

Vitrage also known as artistic stained glass. It was inspired mainly by churches stained glass.

You need Bars Mayhem to run it:
Check out releases on [github](https://github.com/kittysatanicbelyah/satans_deco_bm/releases) or download on [curseforge](https://www.curseforge.com/minecraft/mc-mods/satans-deco-bm).
